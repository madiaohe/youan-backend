# 登录页面设计文档

## 概述

滤盒全生命周期管理系统的登录页面，支持账号密码登录和手机验证码登录两种方式。

## 功能需求

### 登录方式

1. **账号密码登录**
   - 用户名/工号输入
   - 密码输入
   - 记住我选项
   - 登录按钮
   - 忘记密码链接

2. **手机验证码登录**
   - 手机号输入
   - 验证码输入 + 获取验证码按钮（60s 倒计时）
   - 记住我选项
   - 登录按钮
   - 忘记密码链接

### 交互行为

- 使用 Tab 组件切换两种登录方式
- 登录成功后跳转到首页（`/`）
- 点击「忘记密码？」跳转到 `/forgot-password` 页面（后续实现）

### 使用的 shadcn 组件

| 组件 | 用途 |
|------|------|
| `Card` | 登录卡片容器 |
| `Tabs` | 切换登录方式 |
| `Input` | 用户名、密码、手机号、验证码输入 |
| `Button` | 登录按钮、获取验证码按钮 |
| `Checkbox` | 记住我 |
| `Label` | 表单标签 |

**约束：** 必须使用 shadcn 组件库，不允许自定义组件。

## 技术实现

### 页面路由

- 登录页：`/login`
- 登录成功后跳转：`/`（首页/数据看板）
- 忘记密码：`/forgot-password`（后续实现）

### Mock 数据逻辑

- **账号密码登录**：任意用户名 + 任意密码 → 登录成功
- **手机验证码登录**：任意手机号 + 任意 6 位数字验证码 → 登录成功
- **记住我**：状态存储到 localStorage
- **获取验证码**：点击后显示 60s 倒计时，期间按钮禁用

### 状态管理

- 登录状态存储在 localStorage 或 session storage
- 使用 React state 管理表单数据和 UI 状态
