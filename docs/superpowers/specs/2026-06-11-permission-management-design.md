# 权限管理模块设计文档

## 概述

权限管理模块包含管理员列表、角色管理、管理员日志三个子功能，用于管理后台用户的账号、角色权限分配和操作日志追踪。

## 页面结构

### 整体布局

采用左侧 Sidebar 导航 + 右侧内容区的布局结构：

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo] 滤盒全生命周期管理系统                    [用户头像 ▼] │
├──────────────┬──────────────────────────────────────────────┤
│              │                                              │
│  ▼ 权限管理   │          【右侧内容区】                        │
│    管理员列表 │                                              │
│    角色管理   │   根据左侧选中的菜单显示对应内容                  │
│    管理员日志 │                                              │
│              │                                              │
│  □ 员工管理   │                                              │
│  □ 检测管理   │                                              │
│  ...         │                                              │
│              │                                              │
└──────────────┴──────────────────────────────────────────────┘
```

### 路由规划

| 页面 | 路由 |
|------|------|
| 管理员列表 | `/permission/admin` |
| 角色管理 | `/permission/role` |
| 管理员日志 | `/permission/log` |

## 功能详情

### 1. 管理员列表

#### 功能描述

添加、编辑、删除管理员，并给管理员分配对应的操作权限。

#### 字段定义

| 字段 | 类型 | 说明 |
|------|------|------|
| 用户名 | string | 登录账号 |
| 姓名 | string | 管理员真实姓名 |
| 角色 | string | 关联的角色 |
| 状态 | enum | 启用/禁用 |
| 创建时间 | datetime | 账号创建时间 |
| 最后登录 | datetime | 最近一次登录时间 |

#### 页面布局

```
┌──────────────────────────────────────────────────────────────┐
│  管理员列表                                                   │
├──────────────────────────────────────────────────────────────┤
│  [+ 新增管理员]  [批量删除]  [批量分配角色]                     │
├──────────────────────────────────────────────────────────────┤
│  用户名     姓名     角色        状态    创建时间    最后登录    操作  │
│  ────────────────────────────────────────────────────────────│
│  □ admin   张三    超级管理员   启用   2024-01-01  2024-06-11  [编辑][删除]│
│  □ user01  李四    普通管理员   启用   2024-02-15  2024-06-10  [编辑][删除]│
│  □ user02  王五    普通管理员   禁用   2024-03-20  -           [编辑][删除]│
├──────────────────────────────────────────────────────────────┤
│  共 3 条记录                              [< 1 2 3 ... >]     │
└──────────────────────────────────────────────────────────────┘
```

#### 操作说明

- **新增管理员**：弹出 Dialog，填写用户名、姓名、密码、选择角色、设置状态
- **编辑管理员**：弹出 Dialog，修改姓名、角色、状态（用户名不可修改）
- **删除管理员**：弹出 AlertDialog 确认删除
- **批量删除**：勾选多条记录后，点击批量删除按钮
- **批量分配角色**：勾选多条记录后，弹出 Dialog 选择角色

#### 使用的 shadcn 组件

| 组件 | 用途 |
|------|------|
| `Sidebar` | 左侧导航菜单 |
| `SidebarMenu` / `SidebarMenuItem` | 二级菜单项 |
| `Button` | 新增、批量操作、编辑、删除按钮 |
| `Table` | 数据表格 |
| `Checkbox` | 批量选择 |
| `Badge` | 状态标签 |
| `Dialog` | 新增/编辑弹窗 |
| `AlertDialog` | 删除确认弹窗 |
| `Input` | 表单输入 |
| `Select` | 角色选择下拉 |
| `Switch` | 状态开关 |
| `Pagination` | 分页组件 |

### 2. 角色管理

#### 功能描述

创建不同的角色并分配对应的操作权限（菜单级权限）。

#### 字段定义

| 字段 | 类型 | 说明 |
|------|------|------|
| 角色名称 | string | 角色标识 |
| 角色描述 | string | 角色用途说明 |
| 权限数量 | number | 该角色拥有的菜单权限数量 |
| 创建时间 | datetime | 角色创建时间 |
| 状态 | enum | 启用/禁用 |

#### 页面布局

```
┌──────────────────────────────────────────────────────────────┐
│  角色管理                                                     │
├──────────────────────────────────────────────────────────────┤
│  [+ 新增角色]                                                 │
├──────────────────────────────────────────────────────────────┤
│  角色名称      角色描述        权限数量   创建时间    状态    操作    │
│  ──────────────────────────────────────────────────────────── │
│  超级管理员    拥有所有权限      12个     2024-01-01  启用    [编辑][删除]│
│  普通管理员    日常运维管理      8个      2024-01-01  启用    [编辑][删除]│
│  数据查看员    仅查看报表数据    3个      2024-02-15  启用    [编辑][删除]│
└──────────────────────────────────────────────────────────────┘
```

#### 菜单权限列表

基于 Excel 后端功能清单，权限菜单结构如下：

| 一级菜单 | 二级菜单 |
|---------|---------|
| 权限管理 | 管理员列表、角色管理、管理员日志 |
| 员工管理 | 员工信息维护、人脸识别管理、区队管理 |
| 检测管理 | 呼吸阻力检测参数设置、检测记录查询、检测日志导出 |
| 回收管理 | 待回收滤盒数据、回收记录查询、待领用滤盒数据 |
| 设备管理 | 检测回收柜管理、自助发放柜管理、WMS数据同步 |
| 报表与统计分析 | 滤盒检测数据统计、区队不合格率排名、大屏展示配置 |
| 日志管理 | 操作日志、检测与回收日志、AI推荐面罩日志 |
| 系统设置 | 管理员账户管理、设备参数配置、界面与通知配置 |

#### 使用的 shadcn 组件

| 组件 | 用途 |
|------|------|
| `Button`、`Table`、`Badge` | 同管理员列表 |
| `Dialog` | 新增/编辑角色弹窗 |
| `Checkbox` | 菜单权限选择（支持父子联动） |
| `ScrollArea` | 权限列表滚动区域 |
| `Input` | 角色名称、描述输入 |
| `Textarea` | 角色描述输入 |

### 3. 管理员日志

#### 功能描述

查看管理员的系统操作历史记录。

#### 字段定义

| 字段 | 类型 | 说明 |
|------|------|------|
| 操作人 | string | 执行操作的管理员 |
| 操作类型 | enum | 新增/编辑/删除/登录/导出等 |
| 操作模块 | string | 操作所属功能模块 |
| 操作内容 | string | 具体操作描述 |
| 操作时间 | datetime | 操作发生时间 |
| IP 地址 | string | 操作来源 IP |

#### 页面布局

```
┌──────────────────────────────────────────────────────────────┐
│  管理员日志                                                   │
├──────────────────────────────────────────────────────────────┤
│  操作人 [____▼]  操作类型 [____▼]  操作模块 [____▼]           │
│  操作时间 [开始日期] ~ [结束日期]           [查询] [重置]       │
├──────────────────────────────────────────────────────────────┤
│  操作人   操作类型   操作模块   操作内容          操作时间       IP地址    │
│  ─────────────────────────────────────────────────────────────│
│  admin   登录      系统      登录系统成功        2024-06-11 10:30  192.168.1.1 │
│  admin   新增      员工管理   新增员工：张三      2024-06-11 10:25  192.168.1.1 │
│  user01  导出      检测管理   导出检测记录Excel   2024-06-11 09:15  192.168.1.2 │
├──────────────────────────────────────────────────────────────┤
│  共 156 条记录                            [< 1 2 3 ... >]     │
└──────────────────────────────────────────────────────────────┘
```

#### 操作类型枚举

| 类型 | 说明 |
|------|------|
| 登录 | 登录系统 |
| 登出 | 退出系统 |
| 新增 | 新增数据 |
| 编辑 | 修改数据 |
| 删除 | 删除数据 |
| 导出 | 导出数据 |
| 导入 | 导入数据 |

#### 使用的 shadcn 组件

| 组件 | 用途 |
|------|------|
| `Input` | 文本搜索 |
| `Select` | 下拉筛选（操作人、操作类型、操作模块） |
| `Button` | 查询、重置 |
| `Table` | 数据表格 |
| `Pagination` | 分页 |

注：日期选择使用 `<input type="date">` 或 shadcn 内置日期组件。

## Mock 数据逻辑

- **管理员列表**：预设 5-10 条模拟管理员数据
- **角色列表**：预设 3 条角色数据（超级管理员、普通管理员、数据查看员）
- **管理员日志**：预设 20 条模拟日志数据
- **新增/编辑/删除操作**：前端状态变更，Toast 提示成功
- **权限分配**：前端保存权限配置，Toast 提示成功
- **数据持久化**：使用 localStorage 存储数据变更

## 约束

- 必须使用 shadcn 组件库，不允许自定义组件
- 权限粒度为菜单级别（不涉及功能级、数据级权限）
